# Portfolio - OIBSIP Web Development Task 2

## Project Overview

This is a professional portfolio website for **Smit Viramgami**, a Frontend Developer and UI Enthusiast. The project was created as part of **OIBSIP (Oasis Infobyte Summer Internship Program) - Web Development and Designing Task 2**.

The portfolio showcases skills, projects, experience, and provides a way for potential employers and collaborators to get in touch.

---

## Task Description

### Objective
Create a responsive, modern portfolio website that effectively presents:
- Professional profile information
- Technical skills and expertise
- Completed projects and work samples
- Contact information and social links
- Call-to-action elements for hiring opportunities

### Requirements Met
✅ Responsive design that works on all devices (mobile, tablet, desktop)  
✅ Clean and professional UI/UX  
✅ Semantic HTML5 structure  
✅ Modern CSS3 styling with animations  
✅ Smooth navigation and user experience  
✅ Social media and contact integration  
✅ Performance optimized code

## Process Followed

### 1. **Planning & Design**
   - Identified key sections needed for a portfolio website
   - Sketched wireframes for layout and structure
   - Defined color scheme and typography
   - Planned responsive breakpoints

### 2. **HTML Structure**
   - Created semantic HTML5 markup
   - Organized content into logical sections:
     - Header with navigation
     - Hero section with introduction
     - About section with stats
     - Skills section with technology cards
     - Projects section (placeholder for future updates)
     - Services section
     - Contact section
     - Footer with social links

### 3. **Styling with CSS3**
   - Implemented modern CSS3 features
   - Created responsive layouts using Flexbox and Grid
   - Added smooth transitions and animations
   - Optimized for mobile-first approach
   - Implemented custom properties (CSS variables) for consistency

### 4. **Features Implemented**
   - **Navigation Bar**: Sticky navigation with smooth scrolling
   - **Hero Section**: Eye-catching introduction with call-to-action buttons
   - **Stats Section**: Quick metrics about experience and projects
   - **Skills Cards**: Interactive skill showcase with icons
   - **Responsive Design**: Mobile-first approach with media queries
   - **Social Integration**: Links to LinkedIn, GitHub, and Email
   - **Visual Effects**: Hover effects, transitions, and animations

### 5. **Version Control & Deployment**
   - Initialized Git repository
